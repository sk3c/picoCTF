# The Numbers

**Description**
The numbers... what do they mean?

**Vulnerability**
Cryptography

## Solution

**Step 1**
After downloading the image, we can see that it contains a sequence of numbers.
The format of these numbers corresponds to alphabetical indexing, where each number represents a letter of the alphabet (e.g., 1 = A, 2 = B, 3 = C, etc.).

**Step 2**
By converting each number into its corresponding letter, we obtain a readable string, which reveals the flag.
